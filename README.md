# ResumeTake - AI简历优化工具

> 智能AI简历优化，一键生成专业简历，ATS关键词匹配，提升求职成功率。

## 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | SvelteKit + TailwindCSS |
| 后端 | Go (Fiber) |
| 部署 | Docker + Nginx |
| SEO | Schema.org + Open Graph + Sitemap |

## 特性

- AI智能简历优化
- ATS关键词匹配
- 6种专业模板
- 暗色模式
- 极简UI设计
- 完整SEO优化
- Docker一键部署

## 快速开始

```bash
git clone https://github.com/takeqaq2/resumetake.git
cd resumetake
docker compose up -d --build
# http://localhost:8088
```
