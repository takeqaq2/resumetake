# ResumeTake - AI简历优化工具

> 智能AI简历优化，一键生成专业简历，ATS关键词匹配，提升求职成功率。

## 功能

- AI智能简历优化
- ATS关键词匹配
- 专业模板 (6种)
- 暗色模式
- 极简UI设计

## 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | React 18 + TailwindCSS + Vite |
| 后端 | Go (Fiber) |
| 部署 | Docker + Nginx |
| SEO | Schema.org + Open Graph + Sitemap |

## 快速开始

```bash
git clone https://github.com/takeqaq2/resumetake.git
cd resumetake
docker compose up -d --build
# 访问 http://localhost:8088
```
